BUT en Général

Les informations générales

Le Bachelor Universitaire de Technologie (BUT) est un cursus d’une durée de 3 ans qui est reconnu au même titre qu’une licence. Les choix après ce BUT sont donc les mêmes qu’après une licence. Il remplace et améliore le Diplôme Universitaire de Technologie (DUT) qui se faisait en 2 ans.

Le BUT se fait dans un Institut Universitaire de Technologie, le fameux “IUT”. Comme dit récemment, la durée du BUT est de 3 ans donc le diplôme obtenu correspond au niveau d’étude BAC+3.

Que vous veniez d’un BAC général ou d’un BAC technologique, vous pouvez accéder à cette formation sans souci. Si vous ne connaissez pas grand chose en informatique ce n’est pas grave, la première année vous allez voir plein de bases.

La promo est composée d’un peu plus d’une centaine d’élèves qui sont ensuite répartis dans plusieurs groupes.

Vous trouverez le site de l’IUT dans le bas de page, à chaque fois qu’on vous parle d’un site web ou d’une application ou d’un email, les liens se trouvent dans le bas de page, allez y jeter un œil !

Les cours

Il y a 3 types de cours :

  • les cours Magistraux dont le nom fait très sérieux, ce sont des cours avec toute la promotion en amphithéâtre,
  • les Travaux Dirigés, principalement pour apprendre la théorie,
  • Les Travaux Pratiques, vous vous en doutez bien, pour pratiquer la théorie.

Les professeurs

Et non, il n’y a pas qu’un type de professeurs, pour certains celà peut paraître étonnant mais leur objectif reste le même : vous faire apprendre.

Les professeurs de l’IUT, ceux que vous verrez le plus, ils vous enseigneront comme au Lycée, celà ne changera pas vos habitudes.

Les professeurs vacataires, ils vous feront cours normalement mais ce sont des professionnels qui font cours en plus de leur métier, leur pédagogie comparé à des professeurs classiques est plus sympa alors amusez vous bien !

L'administratif

En termes d’administration, les frais d’inscriptions et la CVEC sont gratuits pour les personnes éligibles à la bourse. Pour les non-boursiers il faudra payer.

Pour obtenir une bourse allez voir sur le site du CROUS “Mes services étudiants”, celà vous procure d’énormes avantages.
